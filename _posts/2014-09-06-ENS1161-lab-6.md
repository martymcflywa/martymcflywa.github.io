## Question 19

Find the inverse of each of the following functions

### 1: `f(x) = 5(x − 3)`

```
y = 5(x - 3)
y / 5 = x + 3
y / 5 + 3 = x
y = x / 5 + 3
```

`f`<sup>-1</sup>`(x) = x / 5 + 3`

### 2: `g(x) = 3x + 1`

```
y = 3x + 1
y - 1 = x / 3
(y - 1) / 3 = x
y = (x - 1) / 3
```

`g`<sup>-1</sup>`(x) = (x - 1) / 3`

### 3: `h = √x + 6`

```
y = √x + 6
y^2 = x - 6
y^2 - 6 = x
y = x^2 - 6
```

`h`<sup>-1</sup>`(x) = x^2 - 6`

### 4: `j(x) = (2x + 5) / 3`

```
y = (2x + 5) / 3
3y = (x - 5) / 2
3y - 5 = x / 2
(3y - 5) / 2 = x
y = (3x - 5) / 2
```

`j`<sup>-1</sup>`(x) = (3x - 5) / 2`
