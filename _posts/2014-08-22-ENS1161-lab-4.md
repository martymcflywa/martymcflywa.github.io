### 7: `(y + xz' + x'y')'`

![kmap lab q2-7 3var](http://i.imgur.com/v6fYXRl.png)

#### Issue

Working it out:

```
(y + xz' + x'y')'
y'(x'z + xy)
y'x'z + y'xy
x'y'z + 0
x'y'z
```

#### Correct Answer

```
xy'z
```

But how is the `x` in the answer not a `x'` ?

#### Correct Law

```
(y + xz' + x'y')'
y'(xz')'(x'y')'
y'(x' + z)(x + y)
y'(x'x + x'y + zx + zy)
y'x'x + y'x'y + y'zx + y'zy
0 + 0 + xy'z + 0
xy'z
```
